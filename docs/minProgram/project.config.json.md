# project.config.json

在项目根目录下使用 `project.config.json` 文件对项目进行配置

[微信文档](https://developers.weixin.qq.com/miniprogram/dev/devtools/projectconfig.html)

`miniprogramRoot` -> 指定小程序源码的目录

`cloudfunctionRoot` -> 云函数下主页的路径

`setting` -> 项目设置

- `urlCheck` -> 是否检查安全域名和 TLS 版本

`condition` -> 编译模式配置
